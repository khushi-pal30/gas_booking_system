# Gas Booking System
A django-based web application for managing gas bookings online 
# Features 
-User Registration and login 
-Book gas cylinders 
-View booking history
-admin panel for managing users and booking
## Tech Stack
-Python
-Django
-HTML,CSS,BootStrap
-SQlite(default)
